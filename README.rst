============
PlantScan3D 
============

An open-source editor for reconstructing 3D plant architecture from laser scans.

**PlantScan3D** is released under a **CeCILL** license, which is compatible to the **GNU GPL**.

Installation
============

Requirements
-------------

* OpenAlea.PlantGL

Installation
-------------

<code>
python setup.py install
</code>

Utilisation
-------------

<code>
plantscan3d
</code>
